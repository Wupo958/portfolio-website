Fancy A Kappa - Cozy 3D Game

Fancy A Kappa is an educational cooking simulator created in Unreal Engine with a team of seven. Players interact with mythological NPCs from different cultures, engaging in conversations and preparing traditional dishes. The game combines narrative design, cultural education, and cooking mechanics into a cozy experience. Through this project, I gained experience in Unreal Engine development, gameplay scripting in Blueprints, team collaboration, and designing engaging UI/UX that blends storytelling with game mechanics.

Sportscenter is a VR multiplayer project developed in Unity together with another student. Random User AR App - Mobile Development

This Android app was developed as part of the Mobile Application Development course. It fetches random user data from the randomuser.me API and stores it locally using an SQLite database. Each user is assigned a unique QR code which can be scanned using the device camera. When a known QR code is detected, the associated user information is displayed in real-time through an augmented reality overlay.

Plane Learning to Fly - Machine Learning

A small project where I created autonomous planes in Unity using the PPO (Proximal Policy Optimization) reinforcement learning algorithm. This project gave me hands-on experience with Unity ML-Agents, neural network tuning, and environment design for AI learning.
