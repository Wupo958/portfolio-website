Random User AR App – Mobile Development

This Android app was developed as part of the Mobile Application Development course. It fetches random user data from the randomuser.me API and stores it locally using an SQLite database. Each user is assigned a unique QR code, which can be scanned using the device camera. When a known QR code is detected, the associated user information is displayed in real-time through an augmented reality overlay.

The app features three core screens: a user overview with sorting functionality, a detailed view showing each user's information (including photo, name, birthdate, phone number), and an AR screen for live marker detection and interaction. Additionally, a settings section was implemented to allow database reset and bulk user generation.

Plane learning to fly - Machine Learning

A small project where I created autonomous planes in Unity using the PPO (Proximal Policy Optimization) reinforcement learning algorithm. The planes learned to take off, fly, collect checkpoints, and avoid obstacles through training. This project gave me hands-on experience with Unity ML-Agents, neural network tuning, and environment design for AI learning.

Copper Feathers - 2D Platformer

Copper Feathers is a 2D platformer developed in Unity, created collaboratively with a team of four other students. The game features two distinct levels, a challenging boss fight, upgrade mechanics, environmental obstacles, and fully interactable NPCs. It was awarded Best Mechanics in our course showcase for its polished gameplay systems and intuitive design. This project gave me valuable experience in game design, level scripting, team collaboration, and Unity development.

Arcane Siege - Top-Down Action Adventure

Arcane Siege is a top-down action-adventure game inspired by classic titles like The Legend of Zelda: A Link to the Past. Developed in C++ using the Raylib graphics library, this project was built in collaboration with a team of five. The game features boss fights, environmental puzzles, and three distinct player-controlled spells with different effects and use cases. Creating Arcane Siege deepened my experience in C++ programming, game architecture, and collaborative gameplay design.
